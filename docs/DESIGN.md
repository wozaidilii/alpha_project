# Design

## Visual Direction

AniGuessr should feel like a polished anime location mystery game. Use a dark ink base with pink, cyan, and a small amount of warm accent color. The supplied anime still can be used as a visual texture, but real street view remains the gameplay focus.

## UI Rules

- Homepage: show the playable experience immediately with one primary call to action and a visible language switcher.
- Game screen: keep the anime clue in the upper-left overlay and the guess map as the secondary control surface.
- Do not reintroduce mode cards for historical, battle, foreign, or other legacy modes unless those modes are explicitly reopened.
- Avoid card nesting. Use panels only for clue, result, and compact game controls.
- Keep buttons at least 44px tall, with clear focus states.

## Copy

- Chinese default: `猜动漫模式`, `开始猜动漫`, `返回主页`.
- Japanese and English copy should describe the same rule: real street view plus anime clue, global map guess.
